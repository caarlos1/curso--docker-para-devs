# Docker Hub

## Atuenticando Docker Hub

    docker login

Depois será necessário digitar o docker id e senha.

### Saindo

    docker logout

## Enviando Imagem

    docker push usuario/nomed_da_imagem:tag

Porem antes o repositório tem que ser cadastrado.

## Sincronizando

    docker pull usuario/nomed_da_imagem:tag
